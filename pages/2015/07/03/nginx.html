Nginx Day one

BY Morne Liao

安装

去nginx.org上下载最新的nginx源码包

./configure
make && make install

安装好后测试一下，先修改配置文件，在配置文件中设置服务器根目录的路径为博客的根目录：

root ...

启动nginx，在浏览器中输入localhost:80：
